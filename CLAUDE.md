### Key Data Flow

**Report viewing**: iOS app creates a `share_token` in Supabase â†’ user visits `/report/{token}` or `/student/{token}` â†’ `data_service.validate_token()` checks expiry â†’ data fetched from Supabase â†’ rendered via Jinja2.

**Crypto bot**: APScheduler runs every 10 minutes â†’ `bot_service.py` fetches OHLCV data â†’ `bot_indicators.py` computes RSI/MACD/bias â†’ `bot_ai.py` sends to OpenRouter for AI analysis â†’ Telegram alert dispatched.
